Reallusion USD Sample Characters and Motions - Readme


SAMPLE CHARACTERS AND MOTIONS
================================

Three animated characters in USD format are included. All characters are generated from Character Creator 3 via the Omniverse Connector plug-in, with PBR materials applied:
	* Debra
	* Orc
	* Worker

Special notice: Character Creator Omniverse Connector was developed for Omniverse Create 2021.1. Besides PBR, CC characters can be exported with SSS materials for Path Traced rendering. SSS versions of the aforementioned characters will be provided for Omniverse Create 2021.2 when the following issue will be addressed and resolved by NVIDIA:

*Appearance of excessively bright highlights with the SSS shader, e.g. oily lips.
*Appearance of line segments with the SSS shader, especially on the nose and jawline with the presence of an HDR enabled light dome.

INSTRUCTION
============

Please refer to the following webpage to know how to load and apply freebie characters and motions to Omniverse Create.
https://www.reallusion.com/character-creator/nvidia-omniverse/default.html#tutorials

Special notice: The character should be exported with the animation for the best possible result. For example, animation applied from a standard proportion to a smaller character will give unexpected/erroneous results.
